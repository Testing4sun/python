1. В Python необходимо писать комментарий после символа #

2. Код def capitilize (Принимает на вход текст, делает первую букву заглавной и возвращает этот же текст):
    Код должен быть input[0].upper() + input[1:].lower()

3. Код def to_list (Принимает на вход текст с разделителем и возвращает список строк):
    Код не принимает значение, состоящее только из цифр, на ввод (например, "1234")

4. Код def delete_symbol (Удаляет все подстроки из переданной строки):
    Код принимает пустые значения на ввод как верные

5. Код def starts_with (Возвращает `True`, если строка начинается с заданного символа и `False` - если нет):
    Возвращает символ, а не True/False. Функция не делает ничего, кроме возврата заданного символа

6. Код def list_to_string (Преобразует список элементов в строку с указанным разделителем):
    По умолчанию не стоит запятая с пробелом (", "), как указано в ТЗ
